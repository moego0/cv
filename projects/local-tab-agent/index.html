A repository-aware local AI coding assistant extension for VS Code that executes reasoning locally using local LLMs (via Ollama) and orchestrates web-browser tabs (ChatGPT, Claude, Gemini) via WebSockets to enable advanced planning without cloud API costs.

System Architecture

The extension initializes a local WebSocket server from VS Code. When a prompt is triggered, the agent reads repo context, packages it into a standard JSON payload, and routes it to either a local Ollama model or opens a local browser tab via Puppeteer/user-session. The tab processes the prompt, and the result is piped back through the socket directly to the editor.

Technical Challenges

Synchronizing parallel responses from browser tabs and resolving socket disconnections during large context transmissions. Resolved by implementing chunked data transfers and automatic reconnect handshakes.

Results & Impact

Shipped extension on Open VSX Registry, achieving 650+ active installs. Swarm features let users network old laptops to pool processing memory, allowing 8B LLM models to run with less lag.

Lessons Learned

Learned the intricacies of the VS Code Extension Host process and the constraints of local cross-origin browser WebSockets.
